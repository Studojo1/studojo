# Instagram Carousel Design Document
## Studojo Brand Carousel - 10 Page Design Specification

**Purpose**: Design specification for creating Instagram carousels that explain what Studojo is, using the platform's neo-brutalist design style.

**Dimensions**: 1080x1080px per slide (Instagram square format)

**Target Audience**: Students (college/university students, job seekers, general students)

**Focus**: Value proposition-focused messaging

---

## Design System Reference

### Color Palette

#### Primary Brand Colors
- **Purple**: `#8b5cf6` (Violet-500)
- **Purple Light**: `#dab2ff` (Purple-300)
- **Purple Background**: `#faf5fe` (Purple-bg)
- **Emerald**: `#10b981` (Emerald-500)
- **Emerald Light**: `#5ee9b5` (Emerald-300)
- **Emerald Background**: `#d0fae4` (Green-bg)
- **Yellow**: `#eab308` (Yellow-500)
- **Yellow Background**: `#fefce8` (Yellow-bg)
- **Amber**: `#f59e0b` (Amber-500)
- **Amber Light**: `#fbbf24` (Amber-400)
- **Amber Background**: `#fef3c6` (Orange-bg)

#### Neutral Colors
- **Ink/Text**: `#171717` (Neutral-900)
- **Muted Text**: `#525252` (Muted)
- **Surface/White**: `#ffffff` (Surface)
- **Surface Muted**: `#f5f5f5` (Surface-muted)

#### Accent Colors
- **Pink**: `#ec4899`
- **Pink Light**: `#fda5d5` (Pink-300)
- **Pink Background**: `#fdf2f8` (Pink-bg)
- **Teal**: `#14b8a6`

### Typography

#### Font Families
- **Headings**: Clash Display (Bold, Medium weights)
  - Font family: `"Clash Display", ui-sans-serif, system-ui, sans-serif`
  - Use for: Page titles, major headings, emphasis text
  
- **Body Text**: Satoshi (Regular, Medium weights)
  - Font family: `"Satoshi", ui-sans-serif, system-ui, sans-serif`
  - Use for: Body copy, descriptions, supporting text

#### Typography Scale
- **Hero/Title**: 72-96px (Clash Display, Medium/Bold)
- **Page Heading**: 48-64px (Clash Display, Medium)
- **Subheading**: 32-40px (Clash Display, Medium)
- **Body Large**: 24-28px (Satoshi, Regular/Medium)
- **Body**: 18-20px (Satoshi, Regular)
- **Body Small**: 14-16px (Satoshi, Regular)

### Style Elements

#### Borders
- **Standard Border**: 2px solid `#171717` (Neutral-900)
- **Thick Border**: 4px solid `#171717` (for emphasis)
- **Border Radius**: 
  - Small: `0.5rem` (8px)
  - Medium: `1rem` (16px)
  - Large: `1.5rem` (24px)
  - Extra Large: `2rem` (32px)
  - Pill: `9999px` (fully rounded)

#### Shadows (Neo-Brutalist Style)
- **Small Shadow**: `4px 4px 0px 0px rgba(25, 26, 35, 1)`
- **Medium Shadow**: `6px 6px 0px 0px rgba(25, 26, 35, 1)`
- **Large Shadow**: `8px 8px 0px 0px rgba(25, 26, 35, 1)`

*Note: Shadows create the characteristic "offset" look of neo-brutalist design*

#### Spacing
- **Tight**: 8-16px
- **Standard**: 24-32px
- **Loose**: 48-64px
- **Section Padding**: 64-80px

### Design Principles

1. **Bold & Confident**: Use strong borders, high contrast, vibrant colors
2. **Playful but Professional**: Bright colors with structured layouts
3. **Clear Hierarchy**: Large headings, readable body text
4. **Consistent Shadows**: Always use offset shadows for depth
5. **Rounded Corners**: Soften edges with rounded rectangles
6. **Color Blocks**: Use solid color backgrounds for emphasis
7. **Typography Contrast**: Clash Display for impact, Satoshi for readability

## Design Implementation Notes

### For Canva Creation

1. **Set up Template**: 
   - Create 1080x1080px canvas
   - Set up brand colors as color palette
   - Import or use similar fonts (Clash Display alternative: Bebas Neue, Bungee; Satoshi alternative: Inter, Poppins)

2. **Consistent Elements**:
   - Always use 2px borders in `#171717`
   - Apply shadows consistently (4px, 6px, or 8px offsets)
   - Use rounded corners (1rem to 2rem)
   - Maintain color consistency across pages

3. **Typography Hierarchy**:
   - Largest text: 72-96px (hero)
   - Headings: 48-64px
   - Body: 18-28px
   - Small text: 14-20px

4. **Spacing Guidelines**:
   - Minimum padding: 24px
   - Standard padding: 32-48px
   - Large padding: 64-80px
   - Element spacing: 16-24px

5. **Color Usage**:
   - Use brand colors for backgrounds and accents
   - Maintain high contrast for readability
   - Use white text on colored backgrounds
   - Use dark text (`#171717`) on light backgrounds

6. **Visual Elements**:
   - Keep icons simple and bold
   - Use geometric shapes for decoration
   - Maintain consistent style across all pages
   - Ensure all elements have borders and shadows

### Export Settings

- **Format**: PNG or JPG
- **Dimensions**: 1080x1080px
- **Quality**: High (for Instagram)
- **Naming**: `studojo-carousel-01.png` through `studojo-carousel-10.png`

### Accessibility Notes

- Ensure text contrast meets WCAG AA standards
- Dark text (`#171717`) on light backgrounds (white, light colors)
- White text on dark backgrounds (purple, emerald)
- Avoid yellow text on white (use dark text on yellow)
